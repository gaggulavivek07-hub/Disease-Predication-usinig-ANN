# Disease Prediction App

A Streamlit web app that predicts a likely disease from symptom selections using a simple TensorFlow neural network.

## Features
- Interactive symptom selection with checkboxes
- Neural-network based prediction
- Probability breakdown for candidate diseases
- Sidebar navigation for Home, Prediction, and About pages

## Setup
```bash
cd AIproject
pip install -r requirements.txt
streamlit run App.py
```

## Demo
Run locally at: http://localhost:8501
